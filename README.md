# Krish Pandya - Resume

This repository hosts my latest professional resume via GitHub Pages. 

## About
The public link always points to the most up-to-date version of my resume. 

* **Latest Resume:** [`resume.pdf`](resume.pdf)
* **Live Site:** [https://krish10a.github.io/resume/](https://krish10a.github.io/resume/)

## How to Update
To update the resume, simply overwrite the `resume.pdf` file with the new PDF and push the changes:

```bash
git add resume.pdf
git commit -m "Update resume"
git push
```
The live site will automatically reflect the new version while keeping the same public URL.
